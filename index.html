The service for now works fine up to 35MB files, I'm working on supporting larger sizes.

How does it work?

The connection is made through webRTC connecting the two peers and exchanging information, all the data sent is passed between the peers, a signaling server is used to exchange the connection details, after the connection is done the server doesn't have any connection to the peers messages, the only data kept by the server untill disconnection is the host peer id, securing a private peer to peer connection

None of the ips nor the registers of the users is kept by the server, once you disconnect from the site, the information about your peer is deleted
